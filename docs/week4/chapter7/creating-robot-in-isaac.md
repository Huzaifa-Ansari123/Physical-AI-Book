# Creating Robot in Isaac

Building and importing robot models into Isaac Sim.

## Overview

Isaac Sim supports various robot formats and provides tools for creating new models.

## Import Options

- **URDF Import**: Convert ROS robot descriptions
- **USD Native**: Create directly in Omniverse
- **CAD Import**: From SolidWorks, Fusion 360

## Robot Configuration

- Articulation setup
- Joint drives and limits
- Collision meshes
- Visual materials

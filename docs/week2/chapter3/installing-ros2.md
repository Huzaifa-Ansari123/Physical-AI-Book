# Installing ROS 2

Step-by-step guide to installing ROS 2 on your system.

## Overview

ROS 2 (Robot Operating System 2) is the foundation for modern robotics development.

## Supported Platforms

- Ubuntu 22.04 (Recommended)
- Windows 10/11
- macOS

## Installation Steps

1. Set up your sources list
2. Set up your keys
3. Install ROS 2 packages
4. Environment setup

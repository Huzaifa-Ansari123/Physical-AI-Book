# Understanding Nodes and Topics

Core concepts of ROS 2 communication architecture.

## Overview

Nodes and topics form the backbone of ROS 2's distributed computing model.

## Nodes

- **Definition**: Executable processes that perform computation
- **Purpose**: Modular, reusable components
- **Communication**: Interact via topics, services, and actions

## Topics

- **Definition**: Named buses for message exchange
- **Publishers**: Nodes that send messages
- **Subscribers**: Nodes that receive messages
- **Message Types**: Structured data definitions

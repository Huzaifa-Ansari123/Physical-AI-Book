# Debugging ROS 2 Applications

Tools and techniques for troubleshooting ROS 2 systems.

## Overview

Debugging distributed robotic systems requires specialized tools and approaches.

## Command Line Tools

- `ros2 node list`: List active nodes
- `ros2 topic echo`: Monitor topic messages
- `ros2 service call`: Test services
- `ros2 doctor`: Diagnose system issues

## Visualization

- **RViz2**: 3D visualization
- **rqt**: GUI tools collection
- **PlotJuggler**: Data plotting

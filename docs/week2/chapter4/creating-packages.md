# Creating Packages

Building reusable ROS 2 packages for your projects.

## Overview

Packages are the main organizational unit for ROS 2 code.

## Package Structure

- `package.xml`: Package metadata
- `CMakeLists.txt` or `setup.py`: Build configuration
- `src/`: Source code
- `include/`: Header files
- `launch/`: Launch files

## Creating a Package

```bash
ros2 pkg create --build-type ament_cmake my_package
```

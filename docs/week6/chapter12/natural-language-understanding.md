# Natural Language Understanding

Extracting meaning from spoken commands.

## Overview

NLU interprets user intent from transcribed speech.

## Key Tasks

- **Intent Classification**: What does the user want?
- **Entity Extraction**: Key information pieces
- **Slot Filling**: Parameter identification
- **Coreference Resolution**: Pronoun handling

## Approaches

- Rule-based systems
- Machine learning classifiers
- Deep learning (BERT, transformers)
- Large language models

# Integrating with Robotics

Connecting conversational AI to robot actions.

## Overview

Integration bridges language understanding with physical robot capabilities.

## Integration Points

- **Command Mapping**: NLU to robot actions
- **Feedback**: Robot state to user
- **Grounding**: Language to physical world
- **Safety**: Command validation

## Architecture Patterns

- ROS 2 action servers
- Behavior trees
- State machines
- LLM-based planning

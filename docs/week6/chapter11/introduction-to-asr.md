# Introduction to ASR

Automatic Speech Recognition for robotic systems.

## Overview

ASR converts spoken language into text, enabling voice-controlled robots.

## Key Components

- **Acoustic Model**: Sound to phonemes
- **Language Model**: Phonemes to words
- **Decoder**: Combining models for transcription
- **Feature Extraction**: MFCC, spectrograms

## Popular Systems

- Whisper (OpenAI)
- DeepSpeech
- Vosk
- Kaldi

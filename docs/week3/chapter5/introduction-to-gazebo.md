# Introduction to Gazebo

Getting started with the Gazebo robot simulator.

## Overview

Gazebo is a powerful 3D robotics simulator that integrates with ROS 2.

## Key Features

- **Physics Engines**: ODE, Bullet, DART, Simbody
- **Sensor Simulation**: Cameras, LiDAR, IMU, GPS
- **Robot Models**: URDF/SDF support
- **Plugin System**: Extensible architecture

## Why Gazebo?

- Test algorithms safely before deployment
- Rapid prototyping
- Reproducible experiments
- Cost-effective development

# Creating Robot Models

Building robot descriptions for simulation.

## Overview

Robot models define the physical structure, visual appearance, and behavior of robots in simulation.

## Model Formats

- **URDF**: Unified Robot Description Format
- **SDF**: Simulation Description Format
- **Xacro**: XML macros for URDF

## Model Components

- **Links**: Rigid bodies
- **Joints**: Connections between links
- **Visuals**: Appearance meshes
- **Collisions**: Physics geometry
- **Inertials**: Mass properties

# Plugin Development

Extending Gazebo with custom functionality.

## Overview

Plugins allow you to add custom behaviors, sensors, and controllers to Gazebo.

## Plugin Types

- **World Plugins**: Global simulation control
- **Model Plugins**: Robot-specific behavior
- **Sensor Plugins**: Custom sensor processing
- **Visual Plugins**: Rendering extensions

## Development Workflow

1. Create plugin class
2. Implement required interfaces
3. Build with CMake
4. Load in SDF/URDF
